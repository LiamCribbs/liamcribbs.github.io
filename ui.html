Rocks and Stocks

These interfaces were developed for Rocks and Stocks. I put an emphasis on fun interactions in the windows themselves. For example, instead of pressing a button to open a door, the player must drag a door-shaped slider to its open position. It was a lot of fun experimenting with new interactions, including 2-axis sliders and spinning wheels.

Sky Sailors

I programmed and designed the UI for Sky Sailors. This game has full controller support, so the floating cursor needed to work with controllers as well as mice. I created all of the UI scripts & buttons.

FISHGUN

I programmed and designed the UI for FISHGUN. I added unique animations to each crosshair when firing.
